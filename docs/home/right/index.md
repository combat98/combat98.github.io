# 关于本站

## 📖 文档站简介

南忆文档站是一个全栈编程知识体系的学习平台，致力于为开发者提供系统化、结构化的技术文档和学习资源。

### 🎯 建站目标

- **知识沉淀** - 记录学习过程中的知识点和经验总结
- **体系化学习** - 构建完整的技术知识体系
- **分享交流** - 与更多开发者分享技术心得
- **持续更新** - 跟进技术发展，保持内容的时效性

## ✨ 内容特色

### 全面覆盖
涵盖前端、后端、数据库、中间件、DevOps等多个技术领域，构建完整的全栈知识体系。

### 深入浅出
从基础概念到高级应用，循序渐进，既适合初学者入门，也适合有经验的开发者深入学习。

### 实战导向
注重理论与实践相结合，提供实际案例和最佳实践，帮助快速应用到实际项目中。

### 持续更新
紧跟技术发展趋势，定期更新内容，确保知识的准确性和时效性。

## 🛠️ 技术栈

本文档站使用以下技术构建：

- **框架** - VitePress
- **部署** - GitHub Pages
- **版本控制** - Git

## 📚 内容结构

### 计算机科学基础
- 数据结构与算法

- 操作系统
- 计算机网络
- 计算机组成原理

### 后端开发
- **Java生态** - Spring全家桶、MyBatis等
- **.NET** - ASP.NET Core、Entity Framework等
- **Python** - Django、Flask、数据分析等

### 前端开发
- 基础技术 - HTML、CSS、JavaScript
- 现代框架 - React、Vue、TypeScript
