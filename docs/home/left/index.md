# 快速开始

欢迎来到南忆的全栈编程知识体系文档站！这里汇集了从基础到进阶的各类技术知识。

## 📚 学习路线

### 计算机基础
打好基础是成为优秀程序员的关键：
- **数据结构与算法** - 掌握常用数据结构和算法思想
- **设计模式** - 学习面向对象设计的最佳实践
- **操作系统** - 理解进程、线程、内存管理等核心概念
- **计算机网络** - 掌握TCP/IP、HTTP等网络协议
- **计算机组成原理** - 了解计算机底层工作原理

### 后端开发

#### Java生态
- **JavaSE基础** - Java语言核心特性
- **JVM原理** - 深入理解Java虚拟机
- **并发编程(JUC)** - 多线程与并发工具
- **Spring全家桶** - Spring、SpringBoot、SpringMVC
- **持久层框架** - MyBatis、MyBatis-Plus

#### .NET开发
- **C#语言** - .NET平台核心语言
- **ASP.NET Core** - 跨平台Web开发框架
- **Entity Framework** - ORM框架
- **桌面开发** - WPF、WinForms
- **前端框架** - Blazor

#### Python开发
- **Python基础** - 语法、面向对象
- **数据分析** - NumPy、Pandas
- **Web开发** - Django、Flask
- **自动化脚本** - 提高工作效率

### 前端开发
- **基础三件套** - HTML、CSS、JavaScript
- **TypeScript** - 类型安全的JavaScript超集
- **主流框架** - React、Vue
- **服务端渲染** - Next.js、Nuxt.js
- **构建工具** - Webpack、Vite

### 数据库
- **关系型数据库** - MySQL、SQLServer、Oracle、PostgreSQL
- **非关系型数据库** - Redis、MongoDB、Elasticsearch

### 中间件
- **消息队列** - RabbitMQ、RocketMQ、Kafka
- **分库分表** - MyCat、Sharding Sphere

### DevOps
- **Linux运维** - 系统管理、Shell脚本
- **容器化** - Docker容器技术
- **容器编排** - Kubernetes(K8s)
- **CI/CD** - Jenkins自动化部署

## 🎯 推荐学习顺序

### 初学者路线
1. 选择一门后端语言（Java/.NET/Python）
2. 学习前端基础（HTML、CSS、JavaScript）
3. 掌握一个数据库（推荐MySQL）
4. 学习一个主流框架（Spring Boot/ASP.NET Core）
5. 了解Linux基础和Docker

### 进阶路线
1. 深入学习计算机基础知识
2. 掌握分布式系统和微服务架构
3. 学习消息队列和缓存技术
4. 了解DevOps和自动化部署
5. 持续关注新技术和最佳实践

## 💡 使用建议

- **循序渐进** - 从基础开始，不要急于求成
- **动手实践** - 理论结合实践，多写代码
- **持续学习** - 技术更新快，保持学习热情
- **总结归纳** - 定期回顾和总结所学知识
